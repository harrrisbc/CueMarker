# Welcome to the inbox

This is your dumping ground. Drop half-formed thoughts, voice-to-text rambles, links, and "what if" sparks here.

## How to use it

1. Add a new file in this folder (or ask a Cursor agent: *"add this idea to my notes"*).
2. When something feels real, move it to `notes/ideas/` using the idea template.
3. When an idea becomes a build plan, promote it to `notes/projects/`.

Keep it messy on purpose. Clarity lives in the other folders.
